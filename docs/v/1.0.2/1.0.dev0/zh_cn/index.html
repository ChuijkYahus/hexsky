                                      船只操纵

                                      虽然许多生物性、机械性、媒质融注性构造都能从环境中独立出来，但仍受到诸多物理限制，或是无法具有物理实体。本节中的图案能将世界中的一部分组装为刚性的独立物体，并能控制它们。


                                      这些物体的移动不需外物协助，自此产生了近乎无限的创造性。

                                      船只是独立于世界的、能自由移动的物体。

                                      力是朝向特定方向的驱动。

                                      力矩与力类似，但只涉及旋转。

                                      相对力以船只的轴为参照。

                                      绝对力则以世界的轴为参照。
